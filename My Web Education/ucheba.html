Раскрывающийся список – select

У селекта есть несколько атрибутов: name, size, multiple. Атрибут name работает как у радио-переключателей. Size за количество отображаемых строк списка. В качестве значения туда пижется положительное целое число. Атрибут multiple отвечает за то, можно ли выбрать сразу несколько вариантов из списка (как чекбокс), это атрибут без значения.
Каждый элемент списка выделяется в тег option. Так же, как и с input type="checkbox" и input type="radio" нужно указывать атрибут value со значением, которое будет отправлено на сервер.
